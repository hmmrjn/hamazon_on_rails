# Run Locally

```sh
$ git clone https://github.com/hmmrjn/hamazon_on_rails.git
$ cd hamazon_on_rails
$ bundle install
$ rails db:migrate
$ rails server
# Press Ctrl+C to quit.
```
Open http://localhost:3000/
